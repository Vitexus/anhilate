# Anhilate WebExtension

Anhilate is a Firefox WebExtension that allows you to visually remove any element from a webpage.

## How to Install

1.  Clone this repository to your local machine.
2.  Open Firefox and navigate to `about:debugging`.
3.  Click on "This Firefox" in the left-hand menu.
4.  Click on the "Load Temporary Add-on..." button.
5.  Navigate to the `anhilate` directory and select the `manifest.json` file.

The extension is now installed and ready to use.

## How to Use

1.  Click the Anhilate icon in the toolbar or press `Ctrl+Shift+X` to activate the element selection mode.
2.  The mouse cursor will change to a crosshair.
3.  Hover over any element on the page to see it highlighted.
4.  Click on an element or press `Enter` to remove it.
5.  The element will implode and disappear from the page.
6.  Press `Escape` or click the toolbar icon again to deactivate the selection mode.
